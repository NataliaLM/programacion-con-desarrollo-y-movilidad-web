Departamento de calidad: Necesitamos una aplicación donde podamos localizar a través del código de los faros, trazabilidad, aquellos que se han detectado Como defectuosos, otros Como Mal Funcionamiento y otros para su revisión de calidad. Hemos detectado lo siguiente, a través de código de trazabilidad:


  • Todos los códigos IMPARES No funcionan bien, tienen algún Fallo (Departamento de calidad)
  • Todos los códigos PARES el Defecto supera el 72% de Fallo (Defectuoso)
  • Cada caja consta de 15 faros, cada 3 elementos No funcionaba Ninguno. MÚLTIPLO DE 3 (Desguace-Desperdicio)

Ruego podamos identificar la naturaleza los faros, introduciendo el código del faro.
